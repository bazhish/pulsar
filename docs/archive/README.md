# Arquivo de documentação

Esta pasta guarda relatórios e notas históricas que **não** refletem necessariamente o estado atual do código.

Use a documentação principal em [docs/README.md](../README.md) para setup, arquitetura e operação.

| Arquivo | Descrição |
|---------|-----------|
| `delivery-report-historical.md` | Relatório de entrega de uma fase anterior do MVP |
| `quality-historical.md` | Notas de qualidade e cobertura de uma auditoria passada |

Não versionar aqui logs, coverage HTML ou PDFs gerados em runtime — use `.gitignore` e artifacts do CI.
